# 道路病害智能检测系统

基于多模态大语言模型（MLLM）的道路病害检测与评估系统，遵循《JTG 5210—2018 公路技术状况评定标准》进行路面状况分析。

## 项目概述

本项目利用先进的多模态AI技术，通过图像识别技术自动检测道路病害类型、程度和面积，并计算路面状况指数（PCI）、损坏率（DR）等关键指标，最终给出道路等级评定。

### 主要功能

- 自动识别7种主要道路病害类型
- 计算路面状况指数（PCI）和损坏率（DR）
- 根据国家标准进行道路等级评定
- 提供病害处理建议
- 支持批量图像处理

### 病害类型识别

系统可识别以下7类道路病害：

1. **龟裂** - 按轻度、中度、重度分级
2. **横向裂纹** - 按轻度、重度分级
3. **纵向裂纹** - 按轻度、重度分级
4. **坑槽** - 按轻度、重度分级
5. **麻面** - 按轻度、重度分级
6. **修补** - 一般修补
7. **积水** - 按轻度、重度分级

## 技术架构

- **模型**: 基于Qwen2.5-VL-32B-Instruct多模态模型
- **接口**: OpenAI兼容API接口
- **算法**: 符合JTG 5210-2018标准的PCI计算方法
- **语言**: Python

## 使用方法

### 环境准备

```bash
# 确保安装了必要的Python库
pip install requests base64 json pathlib
```

### 运行程序

修改脚本中的输入路径和输出路径：

```python
folders = [r"你的图像文件夹路径"]
output_json_path = r"输出结果JSON文件路径"
```

执行主程序：

```bash
python PCI_prompt.py
```

### 输出结果

程序会生成包含以下信息的JSON文件：

- `description`: 路面状况简要描述
- `findings`: 检测到的病害详细信息列表
- `total_area_A`: 可见路面总面积
- `DR`: 损坏率
- `PCI`: 路面状况指数
- `grade`: 道路等级（优、良、中、次、差）

示例输出：
```json
{
  "description": "路面整体状况良好，局部有修补痕迹。",
  "findings": [
    {
      "type": "修补",
      "severity": "一般",
      "area": 9.0,
      "length": null
    }
  ],
  "total_area_A": 100,
  "DR": 0.9,
  "PCI": 85.64,
  "grade": "良"
}
```

## 评估标准

### 道路等级划分

- **优**: PCI ≥ 90
- **良**: 80 ≤ PCI < 90
- **中**: 70 ≤ PCI < 80
- **次**: 60 ≤ PCI < 70
- **差**: PCI < 60

### 病害权重设置

系统根据JTG 5210-2018标准设置了不同病害类型的权重：

- 龟裂：轻度(0.6)、中度(0.8)、重度(1.0)
- 横向裂纹/纵向裂纹：轻度(0.6)、重度(1.0)
- 坑槽：轻度(0.8)、重度(1.0)
- 麻面：轻度(0.6)、重度(1.0)
- 积水：轻度(0.6)、重度(1.0)
- 修补：一般(0.1)
